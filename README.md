# doca-installer


please use root to install doca

1- Git clone or download the repository:
```
git clone https://github.com/cpoc-internal/doca-installer
```
Run the shell script to install the doca without accessing the directory, run this as ROOT
```
 chmod +x /root/doca-installer/doca-install.sh
sudo -u root /root/doca-installer/doca-install.sh
```
